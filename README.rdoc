= Benelux v0.5

<b>A mad timeline for your Ruby codes</b>


== Features

* Create timers for any Ruby method
* Store arbitrary messages
* Granular statistics
* Thread-safe

== Example

    require 'benelux'

    # Similar to Benchmark::Tms with the addition of
    # standard deviation, mean value, and total, for 
    # each of the times.
    tms = Benelux.bm(1000000, 5) do
      rand
    end
    
    tms.samples              # => 5
    tms.real                 # => 0.45
    tms.real.sd              # => 0.04
    tms.utime                # => 0.44
    tms.utime.sd             # => 0.02
