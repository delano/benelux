current_track (name=nil,group=nil)

If name is specified, this will associate the current thread with that Track name (the Track will be created if necessary).

If track is nil, it returns the Track object for the Track associated to the current thread.

[show source] 
    # File lib/benelux.rb, line 64
64:   def Benelux.current_track(name=nil,group=nil)
65:     if name.nil?
66:       name = Thread.current.track_name
67:     else
68:       Thread.current.track_name = name
69:       Thread.current.timeline ||= Benelux::Timeline.new
70:       @@mutex.synchronize do
71:         @tracks[name] ||= Track.new(name, group)
72:         @tracks[name].add_thread Thread.current
73:         @reporter.add_thread Thread.current
74:       end
75:     end
76:     Benelux.track(name)
77:   end
